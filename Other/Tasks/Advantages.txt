Преимущества Firesec-2

- Групповой контроллер
- OPC сервер
- Интеграция с ITV
- Интергация с любыми другими сторонними системами
- Интеграция с Алгонт Ассад-М5
- Интеграция с цифровыми сидеокамерами
- Современный интерфейс с более глубокой проработкой смены тем
- Интерфейс под тачскрин. Крупные значки под нажатие пальцем

- Сохранение размеров и положения всех окон
- Сохранение почти всех размеров элементов интерфейса
- Настройка и отображение инструкций в критических ситуациях
- Иерархическая организация графических планов
- Гиперссылки в графических планах
- Более компактный размер планов
- Линии привязки вместо сетки с фиксированным шагом
- Возможность смены типа прибора
- Мониторинг несовпадения нескольких устройств и зон на плане, вместо одного
- При привязке устройства к зоне учитывается тип зоны - охранный или пожарный
- Более понятные шаблоны для параметров конфигурации
- Более гладкое изменение масштаба и позиционирования плана
- Более гладкое изменение масштаба и устройства плана
- Векторная графика у прямоугольников, многоугольников и эллипсов - при изменении масштаба пропорционально изменяется ширина границы
- Возможность задания цвета или карнинки фона.
- Возможность задания фоновой картинки у многоугольника
- Автоматическое распределение зон у страници БИ
- Значки в библиотеке устройств в общем стиле и в соответствие с ГОСТ
- Функции выравнивания по нижнему/верхнему краю в редакторе планов
- Поля вокруг планов при увеличении для того, чтобы можно было выделить элементы, находящиеся в углу
- Возможность сосздания на плане невидимой в ОЗ зоны
- Палитра выбора цвета богаче с возможностью задания собственного цвета
- Более простое и понятное добавление/удаление вершин многоугольника
- Сохранение шаблонов параметров конфигурации в файле конфигурации, а не на сервере
- Слияние нескольких конфигураций в одну
- Задание альтернативных картинок для устройств АМП4, АМ1-О
- В ОЗ все свойства устройств и зон можно открыть по клику, а не из контекстного меню
- Клавиатурное управление сбросом и т.д. в ОЗ
- Функция отключения всех устройств в сработке
- Индикаторы состоянией устройств и их управление из одного места
- Привязка устройства к зоне показывает отдельное окно, вместо выпадающего списка
- Сортировка устройств при добавлении по частоте его использования
- Запуск окна редактирования логики ИУ по нажатию на гиперссылке, а не двойным щелчком
- Выбор чего-то из списка по двойному щелчке, а не только по нажатию кнопки OK
- Возможность сокрытия кладки зоны в ОЗ
- При смене типа устройства показываются значки типа
- Возможность таскания элементов на плане, даже если у них задан прозрачный фоновый цвет
- У выходов в ОЗ нет вкладки Управление
- Отключение/Включение всех устройств в зоне из вкладки зоны, а не только из вкладки Планы
- Возможность задания ширины линий фигур
- При чтении параметров из устройства показывается внятное название, а не номер параметра
- Имитация состояний в ОЗ
- Открытие Администратором файла с конфигурацией при двойном щелчке по нему
- Функция сохранить в файл не спрашивает, в какой файл сохранить, если конфигурация была открыта из файла
- На вкладке Зоны показывается древоводния структура входящих в зону устройств, а не плоская. Работает заметно быстрее и можно водить зажатой мышью для выбора зоны. Нет артефактов при выделении
- Автоподстановка текущей даты в название файла при сохранении в файл
- На вкладке Зоны показывается их значек размещения на плане
- Всплывающие сообщения в панели задач возле часов
- Показ списка последних открытых файлов в панели задач Пуск
- При добавлении устройств есть возможность показать все свободные адреса на шлейфе
