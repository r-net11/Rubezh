- Функция найти объекты должна иметь возможность задания типа переменнтой
- Функция задания значения переменной не позволяет выбрать объектную ссылку
- Не обновляется выбор типа переменной при добавлении переменной
- Функция показа окна с сообщением должна иметь настройку показа в той ОЗ, из которой была вызвана функция
- Функция получения поля объекта не имеет возможности выбрать состояние
- Задание явного значения типа Int в функции Найти объекты заблокировано
