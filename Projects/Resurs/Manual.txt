Работа с библиотекой ResursNetwork
Основной класс для работы NetworksManager. Класс является singleton. Вся работа с удалёнными
устройтсвами производится через него.

1. Методы NetworksManager:

	1.1 Для работы с сетями:

	Добавляет новую сеть (сетевой контроллер) на основет переданного типа
	void AddNetwork(UIDevice.DeviceType);
	
	Удаляет сеть с указанным id из менаджера
	void RemoveNetwork(Guid id);

	Включает или выключает сетевой контроллер 
	void SetStatus(Guid id, bool status)

	Получает состояние сетевого контроллера
	bool GetStatus(Guid id)

	1.2 Для работые с устройствами:

	Добавляет новое устройство в сеть с указанным идентификатором
	void AddDevice(Guid networkId, UIDevice.DeviceType);

	Удаляет устройтсво с указанным идентификатором из сети
	void RemoveDevice(Guid deviceId)

	Включает или выключает сетевое устройтсво 
	void SetStatus(Guid id, bool status)

	Получает состояние сетевого устройтсва
	bool GetStatus(Guid id)

	Читает значение параметра из удалённого устройтсво
	Блокирует вызывающий поток до окончания сетевой транзакции
	Result ReadParameter(Guid deviceId, string parameterName)

	Записывает новое значение параметра у удалённое устройтство
	Блокирует вызывающий поток до окончания сетевой транзакции
	Result WriteParameter(Guid deviceId, string parameterName, ValueType value)

2. События NetworksManager:
	Генерируется при возникновении ошибок в устройтсве или при работе 
	DeviceHasError(Guid deviceId, Error error)

	Генерируется при изменении значения параметра устройтсва
	ParameterChanged(Guid deviceId, string parameterName, ValueType newValue)
	
	Генерируется при изменении состояния устройтсва или сети
	StatusChanged(Guid id, bool newStatus)
